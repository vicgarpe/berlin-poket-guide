## Frases imprescindibles

### Saludos y cortesía

| Alemán | Español |
|---|---|
| Guten Morgen | Buenos días |
| Guten Tag | Buenas tardes |
| Guten Abend | Buenas noches |
| Hallo | Hola |
| Tschüss / Auf Wiedersehen | Adiós |
| Bitte | Por favor / De nada |
| Danke (schön) | Gracias |
| Entschuldigung | Perdona / Disculpa |
| Es tut mir leid | Lo siento |
| Sprechen Sie Spanisch / Englisch? | ¿Habla español / inglés? |
| Ich verstehe nicht | No entiendo |

### En el restaurante

| Alemán | Español |
|---|---|
| Einen Tisch für sechs Personen, bitte | Una mesa para seis, por favor |
| Die Speisekarte, bitte | La carta, por favor |
| Was empfehlen Sie? | ¿Qué recomienda? |
| Ich bin Vegetarier / Vegetarierin | Soy vegetariano / vegetariana |
| Ohne Gluten, bitte | Sin gluten, por favor |
| Das schmeckt sehr gut | Está muy bueno |
| Die Rechnung, bitte | La cuenta, por favor |
| Getrennt / Zusammen | Por separado / Juntos |
| Mit Karte oder bar? | ¿Con tarjeta o en efectivo? |

### En el transporte

| Alemán | Español |
|---|---|
| Wo ist die U-Bahn? | ¿Dónde está el metro? |
| Eine Fahrkarte nach… | Un billete a… |
| Einmal / Zweimal | Una vez / Dos veces (1 / 2 billetes) |
| Einfach / Hin und zurück | Solo ida / Ida y vuelta |
| Wo muss ich umsteigen? | ¿Dónde tengo que transbordar? |
| Ist das der Zug nach…? | ¿Es este el tren a…? |
| Nächste Haltestelle | Próxima parada |
| Ausgang | Salida |
| Eingang | Entrada |

### Emergencias y situaciones difíciles

| Alemán | Español |
|---|---|
| Hilfe! | ¡Ayuda! |
| Rufen Sie die Polizei! | ¡Llame a la policía! |
| Ich brauche einen Arzt | Necesito un médico |
| Wo ist das nächste Krankenhaus? | ¿Dónde está el hospital más cercano? |
| Ich habe meine Geldbörse verloren | He perdido la cartera |
| Ich habe mein Handy verloren | He perdido el móvil |

### Compras

| Alemán | Español |
|---|---|
| Wie viel kostet das? | ¿Cuánto cuesta esto? |
| Das ist zu teuer | Es demasiado caro |
| Haben Sie das in einer anderen Größe? | ¿Lo tiene en otra talla? |
| Kann ich mit Karte zahlen? | ¿Puedo pagar con tarjeta? |
| Haben Sie eine Tüte? | ¿Tiene una bolsa? |
| Geöffnet / Geschlossen | Abierto / Cerrado |
